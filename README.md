## 🚀 Installation & Setup

<div align="center">

### Step-by-Step Installation Process

</div>

### 📥 **Step 1️⃣ Clone Repository**
```bash
$ git clone https://github.com/omkareshwarsinha/Aegis-app-.git
$ cd Aegis-app-
🎉 Repository cloned successfully!
```

### 📂 **Step 2️⃣ Open in Android Studio**
```
1. Launch Android Studio
2. Click → Open
3. Select Aegis-app- folder
4. ⏳ Wait for Gradle sync...
5. ✅ Project ready!
```

### 🔑 **Step 3️⃣ Configure Gemini API**

**Create `.env` file:**
```env
# ═══════════════════════════════════════
# GEMINI AI CONFIGURATION
# ═══════════════════════════════════════
GEMINI_API_KEY=your_api_key_here
API_VERSION=v1
DEBUG_MODE=false

# Get your key from: https://ai.google.dev
```

**Or use template:**
```bash
$ cp .env.example .env
$ nano .env  # Add your API key
```

### 🛠️ **Step 4️⃣ Fix Build Config**

Edit `build.gradle.kts` and remove:
```kotlin
// ❌ Remove this line:
signingConfig = signingConfigs.getByName("debugConfig")
```

### ▶️ **Step 5️⃣ Run the App**
```
1. Connect Android Device OR Start Emulator
2. Click ▶️ RUN button
3. Select Target Device
4. ⏳ App Building...
5. 🚀 AEGIS Launches!
```

## 🐛 Troubleshooting

<div align="center">

### Common Issues & Solutions

</div>

### ❌ **Gradle Build Fails**
```bash
# Solution:
$ ./gradlew clean
$ ./gradlew build
# 💡 Tip: Check internet connection and storage space
```

### ❌ **Gemini API Key Error**
```bash
# Check:
1. ✅ .env file exists in project root
2. ✅ API key is valid
3. ✅ API quota not exceeded
4. ✅ Check: https://ai.google.dev/console
```

### ❌ **App Crashes on Startup**
```bash
# Solutions:
1. Update Android Studio to latest version
2. Clear app cache: Settings → Apps → AEGIS → Clear Cache
3. Rebuild: ./gradlew clean build
4. Check Logcat for detailed errors
```

### ❌ **Emulator Performance Issues**
```bash
# Optimize:
1. Allocate more RAM to emulator (4GB+)
2. Enable GPU acceleration
3. Use x86_64 ABI for better performance
4. Close other applications
```
